## 📞 CÓ CÂU HỎI?

### API Endpoints
```bash
# Xem metrics ngay lúc này
GET http://localhost:3000/api/metrics/concurrent-users

# Track một session (manual test)
POST http://localhost:3000/api/metrics/track-session
Body: { sessionId: "test", currentRoute: "/dashboard" }

# Cleanup sessions (test cron)
GET http://localhost:3000/api/cron/cleanup-sessions?test=true
```

### Database Queries
```sql
-- Xem concurrent users hiện tại
SELECT COUNT(*) FROM session_tracking 
WHERE last_activity > NOW() - INTERVAL '5 minutes';

-- Xem peak hour
SELECT DATE_TRUNC('hour', created_at) as hour, COUNT(*)
FROM session_tracking 
WHERE created_at > NOW() - INTERVAL '24 hours'
GROUP BY DATE_TRUNC('hour', created_at)
ORDER BY COUNT(*) DESC;

-- Xem có slow queries không
SELECT query, mean_exec_time FROM pg_stat_statements 
ORDER BY mean_exec_time DESC LIMIT 10;
```
